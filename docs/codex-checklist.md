# codex.com 動作確認 指示書

目的: React版（GitHub Pages公開版）が **HTML版ではなく**、移植済みの仕様通りに動作するかを確認する。

対象URL:
- https://ysk314.github.io/tap-type-english/

前提:
- 表示がHTML版ではないことを確認してから検証を開始する
  - `view-source:` で `@babel/standalone` が **出ない**こと
  - `/tap-type-english/assets/*.js` が **出る**こと

---

## 0. 事前確認（必須）

1) 対象URLへアクセス  
2) 画面を **強制リロード（⌘+Shift+R）**  
3) `view-source:` でHTMLを確認  
   - OK: `@babel/standalone` がない  
   - OK: `/tap-type-english/assets/` の script がある  
   - NG: まだHTML版 → 検証は中止し、通知

---

## 1. Home画面

確認項目:
- タイトル下に `v2` バッジが表示される
- ユーザー作成ができる（新しい学習者追加）
- 既存ユーザーの選択が反映される
- コース選択で Course 画面へ遷移

判定:
- NGがあればスクショ + 再現手順を記録

---

## 2. Course画面（コース選択後）

確認項目:
- ページ一覧（p30-31 など）が表示される
- セクションカードが表示される
- モードボタンが表示される
- 未解放モードはロック表示

Sランク取得後:
- モードボタンに **ランク色（S/A/B）** が付く
- S取得後に **次モードが解放される**

---

## 3. Play画面（学習開始）

確認項目:
- 開始前に **3秒カウントダウン** が表示される
- カウントダウン時に **音** が鳴る
- タイムバーが表示される
- 残り時間が減少する
- 残り10秒未満でバーが赤くなる

入力判定:
- 正解入力時に効果音（type）
- ミス入力時に効果音（error）+ 画面エラー演出
- 連続ミス2回で次の文字ヒントが表示

音声:
- モード1/2は音声ボタンが動作
- モード3は音声ボタン無効
- 自動再生が動作する（0.5秒後）

---

## 4. Result画面

確認項目:
- Rank（S/A/B/C）表示
- スコア（accuracyScore + timeScore）表示
- 正答率/ミス回数/スコアが表示
- Sランク時に **紙吹雪** 表示
- 効果音（S=fanfare / A,B=success / C=try-again）

判定:
- Sランク時に「次のモードが解放」と表示される

---

## 5. 進捗と解放

確認項目:
- Sランク取得後、Course画面で次モードが解放される
- ランク表示がモードボタンに反映される
- ページ再読み込み後もランクが保持される（localStorage）

---

## 6. 異常系（任意）

確認項目:
- タイムアップ時に結果画面へ遷移する
- タイムアップ時のRank計算が正しくCになる
