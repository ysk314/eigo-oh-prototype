## 5. レッツの理念・価値観（PDF要約）
- 理念の中核: 「子どもの輝く未来に貢献する」
- 教育の核: 子どもの主体性（自走力）を育むこと
- 体験価値キーワード: 「ワクワクと安心」
- 目指す姿:
  - 本物の学力
  - 前向きにやり抜く力
  - 周りに愛される力

### クレド（4つの約束）
1. ワクワクと安心があふれる教室をつくる
2. 一人ひとりの可能性を信じ、らしさを伸ばす
3. 家庭と共に育てる「もう一人の家族」であり続ける
4. 講師自身も学び、挑み、共に育ち続ける

## 6. 教育フレームワーク（内部資料ベース）
### SCALE（主体性ステージ）
- `S` Start（反応段階）
- `C` Choice（選択段階）
- `A` Awareness（気づき段階）
- `L` Link（つながり段階）
- `E` Empower（自走段階）

実務上の目標は、全生徒を`A`以上へ引き上げ、`L`と`E`を増やすこと。

### TAPs（主体性を仕組み化する設計）
- `T` Trigger（やってみたい動機）
- `A` Autonomy（自分で選んだ実感）
- `P` Praise（挑戦・成果の意味づけ）
- `s` See（成長が親に見える）
- `+ Simple`（講師・保護者・生徒の全員が続けやすい）

アプリに落とし込む時は、機能追加よりも「自分で選べる」「成長が見える」「継続しやすい」体験を優先する。

## 12. 市場動向・顧客ニーズ（オーナー提供）
### 市場動向
- 一人当たり教育投資は拡大傾向
- 習い事の低年齢化が進行
- 英語教育の早期化ニーズが高い

### 顧客ニーズ
1. 合格実績より、子どもの成長・相性・主体性重視
2. 英語だけでなく国語力/思考力の同時育成ニーズ
3. 家庭と連携しながら成長が見える指導への期待

## 13. 強み・弱み（オーナー提供）
### 強み
1. 高単価でも選ばれる指導力・ブランド
2. 成長が家庭に伝わる運用（目標設計とLINE連携）
3. 保護者ニーズを新サービスに反映できる組織
4. 創業100年の地域信頼

### 弱み
1. 幼児向けサービスのラインナップ不足
2. 幼児層に対する認知/導線/ブランドの弱さ

## 14. 経営方針・目標（オーナー提供）
1. 幼児向け講座を起点に在籍期間とLTVを最大化
  - 目標ライン: 幼児英語 -> そろばん/速読 -> 中学受検 -> 高校受験
2. 幼児-高校生まで、地域No.1ブランドを確立
  - SNS中心のブランディングを強化

## 15. 絶対NG事項（このプロダクトで守ること）
- 不安を煽る表現・導線
- 生徒を「数字」として扱う打ち出し
  - 合格者数などの数値マーケティングを主軸にしない

### 実装・文言の判断ルール
- OK:
  - 学習の楽しさ、自己効力感、小さな成長の可視化
  - 保護者が「安心」できる進捗共有
  - 生徒が自分で選べる体験設計
- NG:
  - 恐怖訴求（このままだと危険/手遅れ等）
  - 他者比較を煽るランキング設計（人格評価につながる表現）
  - 合格実績のみを前面にした導線

## 16. AI実装時の優先順位
1. 主体性が上がるか（Autonomy/Competence/Relatedness）
2. ワクワクと安心を両立しているか
3. 保護者と教室運営の負担を増やさないか
4. 在籍期間（LTV）に効く導線をつくれているか
5. 数値目標だけでなく、子どもの変化が見える設計か
